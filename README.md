# 📦 File Metadata Microservice

A simple and lightweight Node.js API that returns metadata of an uploaded file.  
Built as part of the freeCodeCamp Back End Development and APIs certification.

---

## 🌐 Live Demo

(https://boilerplate-project-filemetadata-uczp.onrender.com/)


## 🚀 Features

- Upload any file through a simple form
- Get file metadata instantly
- Returns:
  - 📄 File name
  - 🧾 File type (MIME)
  - 📏 File size (in bytes)
